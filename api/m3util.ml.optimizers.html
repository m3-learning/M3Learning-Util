Bases: Optimizer

Implements the Trust-Region Conjugate Gradient (TRCG) optimization algorithm.

TRCG is a second-order optimization method that combines trust-region strategies with conjugate gradient methods to solve optimization problems efficiently, especially for large-scale deep learning models.
